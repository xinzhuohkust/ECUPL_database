基层治理文献摘编数据库

该数据库全面搜集整理古今中外著名政治学者关于国家、阶级、民族、权力、政党、法治、政府、政体、民主政治、政治参与、政治沟通、政治文化、政治发展、基层治理、国家治理等主题的经典论述，形成可供学术引用的论述摘编库，以服务于校内外教师与研究生的教学与研究。为了方便校内外师生查询和引用该数据库的相关资源，提高该数据库的使用效率与社会效益，我们还基于该数据库资源建立了搜索系统。
